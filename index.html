â“ Frequently Asked Questions

Is my password stored anywhere?
No. Your password is NEVER stored. It's used only during encryption/decryption and then discarded. Lost password = Lost encrypted files. This is a security feature, not a bug.
How secure is the encryption?
Very secure! Uses AES-256-GCM (same as banks and military), PBKDF2 with 100,000 iterations for key derivation, and SHA-256 for integrity verification.
Can I use this on iPhone?
No. This tool uses ADB (Android Debug Bridge) which only works with Android devices. iOS has its own ecosystem.
Which Android versions are supported?
Minimum: Android 5.0 (Lollipop) | Recommended: Android 8.0+ | Tested up to: Android 14
